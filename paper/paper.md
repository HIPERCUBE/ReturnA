## 수능 문제 데이터 분류
PDF형태로 수능 문제 데이터에서 문제를 뽑아내고 분류하는 작업이 선행되어야한다.

1. PDF파일을 이미지로 Export해서 문제가 위치한 영역을 판별한다.
 - OpenCV의 Line Detection 기술로 세로로 가장 긴 선분을 탐지한다.
 - 문제 데이터 영역 : 탐지한 세로 영역에 위치한 텍스트
 - 시험 정보 데이터 영역 : 첫페이지에서 탐지한 세로영역 위에 위치한 텍스트
2. 시험 정보 데이터 파싱
 - 문제지 이름 : `\d{4}학년도.+문제지`
 - 문제지 유형 : `(A|B)형`
3. 시험 문제 데이터 파싱
